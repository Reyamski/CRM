## What You Can Do

### Dashboard
- See how many active and pending client files you have
- See recently updated client records
- Check your access level and permissions

### Clients
- View all client records in a table
- Search by name, phone, email, or address
- Filter by status (Active, Pending, Closed, Archived)
- Click a client row to open their full details on the right
- Add a new client using the **Add client** button
- Edit an existing client using the **Edit client** button

### Client Detail (right panel)
- View all client info: name, DOB, email, phone, address, marital status, file location
- View the masked SIN (e.g. `***-***-284`)
- If you have permission, click **Reveal SIN** to see the full number — this is logged automatically
- Add timestamped notes at the bottom

### Audit Log
- Only visible to Owner and Manager roles
- Shows a full history of all actions: who did what and when
